## **Dashboard para a área de Recursos Humanos**
 - **Esse dashboard oferece uma visão detalhada da força de trabalho da empresa, permitindo que os gestores de RH monitorem o desempenho, identifiquem áreas de oportunidade e tomem decisões para otimizar a gestão de talentos. Além disso, os filtros permitem aos usuários segmentar os dados e analisar tendências dentro desses grupos demográficos, o que pode ser útil para identificar possíveis disparidades ou áreas de melhoria em termos de diversidade, equidade e inclusão.**
